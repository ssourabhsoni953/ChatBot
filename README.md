## ✨ Features

- **💾 Persistence:** Seamlessly saves chat history to your browser's Local Storage so you never lose a conversation.
- **📜 Auto-Scroll:** Intelligently auto-scrolls to the newest messages as they arrive.
- **📱 Responsive Design:** Flawless experience across desktop, tablet, and mobile devices.
- **🎨 Clean UI:** A minimalist, aesthetic, and highly intuitive user interface.

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Vite
- **State & Refs:** React Hooks (`useState`, `useEffect`, `useRef`)
- **Styling:** Vanilla CSS
- **Deployment:** Vercel

## 🚀 Local Setup

To get a local copy up and running, follow these simple steps:

### Prerequisites

Make sure you have Node.js and npm installed on your machine.

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/repo-name.git
   ```

2. **Navigate to the project directory**

   ```bash
   cd repo-name
   ```

3. **Install dependencies**

   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```
